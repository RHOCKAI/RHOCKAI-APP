# Sound Assets

This directory should contain sound effects for the AI Workout Tracker.

## Recommended Files
- `ready.wav`: Countdown start
- `rep_count.wav`: Successful rep
- `posture_alert.wav`: Incorrect posture warning
- `workout_complete.wav`: Session end

## Status
Currently, the app uses `flutter_tts` for voice feedback. These sound files can be used to add high-quality cues.
