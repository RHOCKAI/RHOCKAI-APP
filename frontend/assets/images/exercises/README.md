## 🎨 Style Guide
- **3D rendered** medical-grade visualization
- **Neon cyan (#00D9FF)** and **purple (#9D4EDD)** lighting
- **Dark background** (#0F0F23) with grid
- **Glowing skeleton overlay** showing joint alignment
- **Dimensions**: 1200x800 (4:3 ratio)
- **Format**: PNG, <500KB per file
